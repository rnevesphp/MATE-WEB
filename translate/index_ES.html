HTML

HTML es el componente básico de la web, permite insertar contenidos y establecer la estructura básica de un sitio web. Por lo tanto, sirve para dar sentido y organizar la información de una página en el página web. Sin ella, el navegador no sabría cómo mostrar texto como elementos o cargar imágenes y otros contenidos.
